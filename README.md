Stack utilisée :
Frontend :
Vue.js 3 (Composition API)
Vite (serveur de développement rapide)
CSS3 (responsive + mobile-first)
Fetch API

Backend :
Node.js
Express.js
CORS

Stockage côté client :
IndexedDB :
stockage structuré des cartes
persistance locale
support du mode hors ligne

Installation et lancement
Frontend
npm install
npm run dev
Le frontend démarre généralement sur : http://localhost:5173

Backend
Dans le dossier backend :
npm install
node server.js
Le backend démarre sur :http://localhost:3000

API disponible ici : http://localhost:3000/api/data
